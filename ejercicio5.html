Tipos de Sistemas Operativos

Linux, Windows, Mac son los más utilizados a día de hoy

Linux Windows Mac Unix
  1. Sistemas operativos Libres
  2. Sistemas operativos Propietarios
  3. Sistemas operativos de código abierto

Linux

Es un sistema operativo libre, por tanto podemos acceder a su código fuente.

Windows

Es un sistema operativo propietario, por tanto es de pago y no podemos acceder a su código fuente.

Mac

Es desarrollado y administrado por Apple, es un sistema operativo propietario.

Unix

Un sistema operativo poderoso utilizado principalmente en servidores y estaciones de trabajo. macOS tiene una base Unix.
